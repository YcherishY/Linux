# 一.递归函数
## 思路

1.先定义递归函数<br>
2.判断输入参数，若没有输入给出提示<br>
3.判断参数是否等于1，不等于继续执行<br>
4.若大于1递归计算n！<br>
5.输出阶乘结果<br>
